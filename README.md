# 🛒 ECommerceApp

This project is a simple e-commerce system developed in C# using .NET.  
It includes intentional bugs and test cases designed to detect these issues.

---

## 📦 Project Structure

ECommerceApp/
├── Core/
│   ├── Product.cs
│   ├── Cart.cs
│   ├── OrderService.cs
│
├── Tests/
│   ├── UnitTests/
│   ├── IntegrationTests/
│
├── Program.cs
└── TestReport.md

---

## ⚙️ Features

- Product selection
- Add to cart
- Order placement
- Payment processing

---

## 🐞 Known Bugs (Intentional)

- Discount is applied incorrectly in all cases
- Null cart is accepted during order
- Negative payment values are allowed

---

## 🧪 Testing

This project uses **NUnit** for testing.

- ✅ Black Box Testing

- ✅ Integration Testing
